# Построитель дерева Штейнера с помощью эвристики iterated-1-steiner ($L_1$ метрика)

Реализация итеративного алгоритма построения дерева Штейнера на сетке Ханана.
Поддерживаются два алгоритма:

- **базовый** – перебирает всех кандидатов из полной сетки Ханана, для каждого кандидата перестраивает MST и добавляет наиболее выгодную точку Штейнера на каждой итерации;
- **оптимизированный (`-m`)** – использует геометрическую фильтрацию кандидатов (edge boxes), динамическое обновление MST и приоритетную очередь для ускорения.

## Требования

- Python 3.8+
- numpy
- matplotlib

## Установка

```bash
git clone https://github.com/ErmolaevDmitrii/vlsi_cad_course_iterated_1_steiner.git
cd vlsi_cad_course_iterated_1_steiner
chmod +x steiner.py
```

## Использование

```bash
./steiner.py input.json [-m] [-v] [--check] [-o output.json]
```

Аргументы:
- `input.json` – файл с вершинами;
- `-m`, `--modified` – использовать оптимизированный алгоритм;
- `-v`, `--visualize` – сохранить изображение дерева в `<script_dir>/<имя>_out.png`;
- `--check` – проверить степень точек Штейнера и число рёбер после построения;
- `-o`, `--output` – указать имя выходного JSON (по умолчанию `<script_dir>/<имя>_out.json`).
